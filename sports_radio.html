Sports Radio

The objective of this design exercise is to create a user interface for an Apple CarPlay app.

Outline

Design Rationales Mid-Fi Wireframe High-Fi Wireframe

Design Rationales

A CarPlay app must be safe to use while driving. Its UI must be concise and clear without creating distractions.

Sports Radio is an app for sports lovers on the road. Sometimes they may not have time to enjoy live games when they are driving and watching the live stream on mobile phones while driving is distractive and extremely dangerous.

Sports Radio offers them an audio version live stream of the games that they don’t want to miss. This app includes all kinds of sports, and the user can follow their favourite teams.

When no live game is going on, the user could also LISTEN to news about their favourite teams or leagues.

Mid-Fi Wireframe

Overall Structure

  • Home screen

    • In this “Leagues” tab view, different league logos are shown as buttons. Under “My Teams” tab, there will be a list of teams that the users created on the mobile phone version of this app. “Trending” tab will include important matches that are being broadcast.

  • Game list

    • Game list view, “Live Games” tab is active. Users can also tap on the “News” tab to see the list of news or the "Programs" tab to listen to the sports commentating programs they like.

  • Live game

    • Live game screen. Team logos will be shown.

High-Fi Mockup

Overall Structure

  • CarPlay home page
  • Home screen

    • Logos of different sports leagues are added to the buttons.

  • Game list

    • Since all games listed here are live, the scheduled date and game time in the mid-fi wireframe is removed. Instead, the list will show the current state.

  • Live game

    • Team logos, names and basic game information are provided. The background is a blurred picture of the home team’s court, indicating the location of the game.

Alternative inputs

  • Control Knob

    • Not all cars' infotainment systems​ are equipped with touch screens. Some of them still rely on physical gadgets as input devices. The UI of this CarPlay app also has to accommodate such needs. Here is a demonstration showing how it works when the user turns the knob clockwise.
